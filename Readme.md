# Microservice practice

## Description

Implements a **Car Rating** system. It contains the folling three (3) microservices:
- Catalog 
- CarInfo
- Ratings

![Logic](Plan.JPG)


## Build process

Go in the corresponding directory for each module and execute 
**mvn package**
